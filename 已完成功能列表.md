## 🚀 如何使用

### 启动后端
```bash
cd backend
python setup_database.py  # 首次运行初始化数据库
python main.py
```

### 启动前端
```bash
cd frontend
npm install  # 首次运行安装依赖
npm run dev
```

### 访问系统
- 前端: http://localhost:5173
- 后端: http://localhost:8000
- API文档: http://localhost:8000/api/docs
