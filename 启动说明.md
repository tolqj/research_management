     ```json
       "password": "admin123"
     }
     ```

## 🔧 问题排查

### 前端依赖安装时间较长
- 首次安装需要下载约 200+ MB 的依赖包
- 预计耗时: 3-10 分钟（取决于网络速度）
- 如果超过 10 分钟，可以在命令行窗口按 Ctrl+C 取消，然后重试

### 前端启动失败
手动启动方式：
```bash
cd "e:\qoder\科研管理系统\frontend"
npm install
npm run dev
```

### 后端服务异常
重启后端：
```bash
cd "e:\qoder\科研管理系统\backend"
python main.py
```

## 📊 系统功能

### 已实现的核心功能
- ✅ 用户认证与权限管理
- ✅ 项目管理（CRUD + 状态流转）
- ✅ 论文管理（JCR/中科院分区）
- ✅ 经费管理（支出记录）
- ✅ 成果管理（专利/奖项/著作/软著）
- ✅ 统计分析（图表可视化）
- ✅ 操作日志

### 数据库表结构
- `users` - 用户表
- `projects` - 项目表
- `papers` - 论文表
- `funds` - 经费表
- `achievements` - 成果表
- `operation_logs` - 操作日志表
