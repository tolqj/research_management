# 快速开始指南

> 5分钟快速部署科研管理系统

## 📋 前置条件

在开始之前，请确保您的系统已安装以下软件：

| 软件 | 版本要求 | 下载地址 |
|------|---------|---------|
| Python | 3.13+ | https://www.python.org/downloads/ |
| Node.js | 18.x+ | https://nodejs.org/ |
| MySQL | 8.0+ | https://dev.mysql.com/downloads/mysql/ |

### 快速检查环境

```bash
# 检查Python版本
python --version

# 检查Node.js版本
node --version

# 检查MySQL是否运行
mysql --version
```

## 🚀 一键启动（推荐）

### Windows系统

1. **启动MySQL服务**
   - 打开"服务"管理器（Win+R 输入 `services.msc`）
   - 找到MySQL服务，确保状态为"正在运行"

2. **配置数据库连接**
   - 打开 `backend\database.py`
   - 修改以下配置（根据您的MySQL实际配置）：
   ```python
   MYSQL_USER = "root"         # 您的MySQL用户名
   MYSQL_PASSWORD = "root"     # 您的MySQL密码
   MYSQL_HOST = "localhost"
   MYSQL_PORT = "3306"
   ```

3. **初始化数据库**
   ```bash
   cd backend
   python setup_database.py
   ```
   
   看到以下提示表示成功：
   ```
   ✓ 数据库初始化完成！
   默认账号信息：
   管理员: admin / Admin@123
   ```

4. **一键启动**
   - 双击运行 `start_all.bat`
   - 等待系统自动启动后端和前端服务

5. **访问系统**
   - 浏览器打开：http://localhost:5173
   - 使用默认账号登录

### Linux/Mac系统

1. **配置数据库连接**
   ```bash
   vi backend/database.py
   # 修改数据库配置
   ```

2. **初始化数据库**
   ```bash
   cd backend
   python3 setup_database.py
   ```

3. **启动后端**
   ```bash
   # 安装依赖
   cd backend
   pip3 install -r requirements.txt
   
   # 启动服务
   python3 main.py
   ```

4. **启动前端（新终端）**
   ```bash
   # 安装依赖
   cd frontend
   npm install
   
   # 启动服务
   npm run dev
   ```

5. **访问系统**
   - 浏览器打开：http://localhost:5173

## ❓ 常见问题

### 1. 数据库连接失败

**现象**：启动时提示 `Can't connect to MySQL server`

**解决方案**：
```bash
# Windows：启动MySQL服务
net start MySQL80

# Linux：启动MySQL服务
sudo systemctl start mysql

# 检查MySQL状态
mysql -u root -p
```

### 2. 端口被占用

**现象**：`Address already in use: 8000`

**解决方案**：
```bash
# Windows：查找占用进程
netstat -ano | findstr "8000"
taskkill /F /PID <进程ID>

# Linux：查找并终止进程
lsof -i :8000
kill -9 <PID>
```

或修改端口配置：
- 后端：修改 `backend/main.py` 第62行的 `port=8000`
- 前端：修改 `frontend/vite.config.js` 第8行的 `port: 5173`

### 3. 依赖安装失败

**现象**：`pip install` 或 `npm install` 报错

**解决方案**：
```bash
# Python使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# Node.js使用国内镜像
npm config set registry https://registry.npmmirror.com
npm install
```

### 4. 前端白屏或无法访问后端

**现象**：页面空白或提示网络错误

**解决方案**：
1. 确认后端服务已启动（访问 http://localhost:8000）
2. 检查浏览器控制台的错误信息
3. 清除浏览器缓存，强制刷新（Ctrl+Shift+R）

### 5. 密码不符合要求

**现象**：登录时提示密码过期或强度不足

**说明**：系统符合等保二级标准，密码要求：
- 至少8位
- 包含大写字母、小写字母、数字、特殊字符
- 90天需要更换

**解决方案**：使用"忘记密码"功能或联系管理员重置
